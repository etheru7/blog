Install PHP dan apache web server di android

install apache web server di android ini adalah solusi yang saya gunakan untuk belajar web programming, karena saya tau tidak semua orang memiliki laptop/pc untuk memulai belajar web programming. Selalu banyak cara untuk tetap belajar.

Aplikasi yang digunakan :
- Termux (bisa download di playstore)
- Browser

Step 1 - Install PHP dan apache
untuk install PHP dan apache, bisa ketikan aja command dibawah ini Selanjutnya langsung enter saja untuk melanjutkan instalasi, jika ada opsi pilihan "y/n" ketikan saja "y".
Step 2 - Test instalasi
untuk test instalasi apakah berhasil terinstall. pertama kita cek versi php. Setelah run command di atas, termux akan menampikan versi dari PHP yang kita install. sekarang kita coba jalankan PHP nya. kita akan buat file .php untuk mencoba nya, tuliskan command di bawah untuk membuat file baru di folder sdcard/htdocs, kalian bisa cek folder htdocs nya di internal storage. Selanjutnya kita start Apache web server. Ketik command ini di termux, lalu enter untuk eksekusi command-nya Setelah melakukan command diatas, termux akan otomatis membuka browser dan mengakses url localhost http://locahost:8080 dan jika muncul tampilan seperti dibawah tandanya kita sudah berhasil menginstall php dan apache web server nya sampai sini kita sudah selesai installasi php dan apache nya.
